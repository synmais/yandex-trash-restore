# Yandex Trash Restore

		Зачем?
		
		Поскольку функции "Восставновить все" в корзине Яндекс Диска нет, 
	то в случае необходимости восстановления сотен, тысяч и даже десятков тысяч 
	файлов не обойтись без стороннего решения.
		Данный скрипт помогает восстановить все файлы из корзины Яндекс Диска 
	при помощи браузерных запросов - в точности, как если бы вы листали корзину 
	и восстанавливали файлы вручную.
	
	
		Как?
		
		1. Откройте браузер и перейдите в свой Яндекс Диск
		2. Перейдите в режим WebDev (F12), выберите вкладку "Сеть" ("Network")
		3. Совершайте любые действия (например, перейдите в Корзину, восстановите 
	какой-нибудь файл вручную) пока в WebDev не увидим запрос типа "models-v2?m=..."
	(например, "models-v2?m=mpfs/bulk-async-restore")
		4. Нажмите правой кнопкой мыши на запросе "models-v2?m=..." и выберите 
	"Скопировать" ("Copy") --> "Копировать как PowerShell" ("Copy as PowerShell")
		5. Откройте файл session.txt и вставьте туда содержимое буфера (как есть, 
	ничего удалять не нужно)
		6. Запустите командную строку (PowerShell) и выполните ".\restore.ps1"
		
		В н и м а н и е - файл session.txt содержит ваши личные данные, не публикуйте его!
		
		
		ЧАВО:
		
		1.	В: Какое ПО необходимо?
			О: PowerShell 5.1 (уже предустановлен на современных версиях Windows) или новее
		
		2.	В: Отсутствует session.txt
			О: Переименуйте session.example.txt в session.txt
		
		3.	В: Восстановление прервано. "Ошибка получения списка файлов"
			О: Данные сессии устарели. Повторите шаги 1-5 инструкции
