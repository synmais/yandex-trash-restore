Краткая инструкция:
1. Перейдите в свою корзину Яндекс Диска
2. Перейдите в WebDev (F12), вкладка "Сеть" ("Network")
3. Выберите любой файл и нажмите "Восстановить"
4. В WebDev нажмите правой кнопкой мыши на models-v2?m=mpfs/bulk-async-restore и выберите "Скопировать" --> "Копировать как PowerShell" ("Copy" --> "Copy as PowerShell")
5. Открываем файл Cookies.txt и вставляем туда содержимое буфера как есть - ничего удалять не нужно
6. Запускаем командную строку, командой pwsh переходим в PowerShell 7 и запускаем скрипт Restore-YandexTrash.ps1
7. Скрипт начнет восстановление всех удаленных файлов
